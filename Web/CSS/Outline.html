Outline

Es un shothand

Las propiedades del outline son:

  1. Width
  2. Style
  3. Color
  4. Offset

Existen bastantes combinaciones:

  • outline-width: 2px; (top, right, bottom, left).
  • outline-style: solid dotted; (X, Y).
  • outline-color: red blue green; (top, X, bottom).
  • outline-width: 2px 3px 4px 5px; (top, rigth, bottom, left).
  • outline-offset: +-px|em|rem...

Propiedades soportadas:

outline-width:

px, em, rem, %.

outline-style:

none, hidden, dotted, dashed, solid, double, groove, ridged, inset, outset, intial, inherit.


Estas medidas dependen del navegador:


thick (ancho), medium (medio), thin(fino).

outline-color:

css native, rgb, rgba, hsl, hsla, hexadecimal, transparent.

Combinaciones:

  • outline-top-(width, style, color).
  • outline-rigth-(width, style, color).
  • outline-bottom-(width, style, color).
  • outline-left-(width, style, color).
