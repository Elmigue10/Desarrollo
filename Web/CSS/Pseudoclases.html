Pseudoclases

Son selectores que reaccionan en tiempo real detectando la interaccion del usuario con algunos elementos.

Funcionan con todos los selectores de CSS

No estan atados a ningun elemento

Su sintaxis es:

selector:pseudoclase{
estilos
}
:pseudoclase{
estilos
}

Pseudoclases 2

Child:

Selecciona un hijo.

  • :first-child; Selecciona al primer hijo
  • :last-child; Selecciona al ultimo hijo
  • :nth-child(odd); Selecciona los hijos impares
  • :nth-child(even); Selecciona los hijos pares
  • :nth-child(n); Selecciona (bn) hasta llegar a b.
